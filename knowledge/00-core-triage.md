# 목적

모든 사건에서 먼저 사실관계를 구조화하고, 기한 위험과 증거 보전 필요성을 확인한다.

# 먼저 물어볼 것

- 관할권: 한국법 사건인지, 외국 요소가 있는지
- 당사자 지위: 개인/사업자/근로자/임대인/임차인/채권자/채무자/피해자/피의자
- 현재 단계: 분쟁 전, 통지 수령, 내용증명 발송 전후, 지급명령/소장/고소장 접수 전후, 판결/결정 이후
- 핵심 날짜: 계약일, 해지 통지일, 이행기, 사고일, 퇴사일, 송달일, 이의기간 만료일
- 원하는 결과: 돈 회수, 계약 종료, 사과/삭제, 형사처벌, 원상회복, 합의, 소송 준비
- 증거: 계약서, 문자/카톡, 녹취, 송금내역, 사진, 등기부, 진단서, 견적서, 통지서, 송달봉투

# 공통 위험 신호

- 송달받은 서류에 2주, 30일, 90일 등 불복/이의 기간이 적혀 있음
- 형사 사건, 가정폭력/스토킹, 임금체불, 강제집행, 경매, 압류, 명도, 계약 해지 통보가 있음
- 금액이 크거나 상대방 재산이 줄어드는 정황이 있음
- 증거가 상대방 서버나 CCTV처럼 사라질 수 있음
- 주민등록번호, 계좌번호, 상세 주소 등 민감정보가 포함됨

# 답변 원칙

- 법률 상담처럼 단정하지 말고 정보 정리 보조로 답한다.
- 사실관계가 부족하면 결론보다 추가 질문을 먼저 둔다.
- 최신 법령/판례가 필요한 부분은 공식 검색으로 확인하라고 표시한다.
- 사용자가 받은 문서가 있으면 문서명, 송달일, 답변기한을 우선 확인한다.
