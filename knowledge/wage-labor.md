# 핵심 판단 흐름

임금체불 사건은 근로자성, 미지급 금품의 종류와 금액, 퇴사일 또는 지급기일, 사업장 정보, 증거를 먼저 확인한다.

# 먼저 물어볼 것

- 근로계약서, 실제 근무시간, 업무지시 방식
- 미지급 항목: 월급, 주휴수당, 연장/야간/휴일수당, 연차수당, 퇴직금
- 재직 중인지 퇴사했는지, 퇴사일과 마지막 지급일
- 급여명세서, 통장입금내역, 출퇴근 기록, 카톡/업무지시
- 4대 보험 가입 여부와 사업자등록 정보
- 해고 통보일, 해고 사유, 서면 통지 여부

# 핵심 쟁점

- 임금체불은 임금지급 의무나 금품청산 의무 위반과 관련된다.
- 근로자는 지방고용노동관서에 진정 또는 고소를 할 수 있다.
- 부당해고는 노동위원회 구제신청 기한 등 별도 기한이 중요하다.

# 대응 방향

- 미지급 내역을 월별/항목별 표로 정리
- 사업주에게 지급 요청 메시지 또는 내용증명 발송
- 고용노동부 진정/고소, 체불임금확인서, 민사 청구 가능성 검토
- 해고 사건은 통보일 기준 기한을 즉시 확인

# 주의

노동 사건은 기한과 증거가 중요하다. 해고, 산재, 임신/육아, 직장 내 괴롭힘이 섞이면 상담 필요성이 높다.
